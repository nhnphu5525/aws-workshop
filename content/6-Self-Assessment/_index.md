My journey at **First Cloud Journey (FCJ)** spanning from **September 6, 2025** to **December 15, 2025** has been a transformative experience. Over these months, I gained hands-on exposure to Amazon Web Services, contributed to a collaborative team project, and learned valuable lessons about working in a professional tech environment.

### Summary of Activities

The program was structured around two parallel learning tracks:

**AWS Technical Training:**
- Successfully completed all 6 AWS Modules encompassing core areas: fundamentals, networking, compute, storage, security, and databases
- Executed 33 practical lab exercises ranging from foundational EC2 instances to sophisticated implementations using CDK, SageMaker, and CloudWatch monitoring
- Participated in 5 community events such as Vietnam Cloud Day, GenAI Builder Club sessions, and the Well-Architected Security workshop

**Collaborative Project - MapVibe:**
- Contributed to a 5-member team developing a location-based restaurant discovery platform
- Led the database architecture using PostgreSQL with PostGIS and pgvector extensions
- Engineered a dual-mode search system integrating BM25 keyword matching with vector-based semantic search
- Implemented RAG (Retrieval-Augmented Generation) capabilities for conversational queries
- Orchestrated cloud migration using Terraform for Infrastructure as Code

This experience strengthened my capabilities in cloud solution design, IaC practices, serverless computing, and collaborative software development with defined milestones.

### Evaluation Rationale

**Technical competence (Good):** Mastered 33 lab exercises and translated theoretical knowledge into a production-ready search engine. My growth from understanding basic VPC concepts to deploying complex RAG architectures with Terraform demonstrates substantial development.

**Learning agility (Good):** Rapidly acquired proficiency in emerging technologies including pgvector for similarity search, hybrid retrieval algorithms, and RAG system design - all accomplished within the internship timeframe.

**Self-initiative (Good):** Proactively investigated technology alternatives, recommended architectural patterns, and sought solutions independently before escalating issues.

**Time management (Good):** Maintained punctuality, respected schedules, and delivered on commitments. Demonstrated adaptability when team meetings were rescheduled while ensuring personal deliverables remained on track.

**Communication skills (Fair):** Showed gradual improvement throughout the program, though I recognize the need to enhance technical presentation clarity and establish more proactive stakeholder updates.

**Analytical thinking (Fair):** Successfully resolved technical issues, though resolution time could be optimized. Developing a more methodical troubleshooting framework remains a priority.
